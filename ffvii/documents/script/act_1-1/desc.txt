scene.xml    contain list of objects that needs to be created during load of field    def types:        * ScriptEntity - just script. Run "on_start" and then "loop".        * ModelEntity - just model.        * ModelControlledEntity - model that iteract with walkmesh and can be controlled.        * TriggerLine - trigger in form of line that send events to script.        * SpawnPointXYTriangle - data about point and triangle. Can be used to spawn or teleport something to this point.        * SpawnPointXYZ - data about point. Can be used to spawn or teleport something to this point.        * Path - set of points. ModelControlledEntity can be set to move by this points one by one.
